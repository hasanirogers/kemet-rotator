# KemetRotator

A component that rotates through an array of text.


## How to use

### Installation

```bash
npm install @kemet/kemet-rotator
```

```js
import '@kemet/kemet-rotator/kemet-rotator.js';
```

### Basic Usage
```html
<kemet-rotator messages='["my", "text", "here"]'></kemet-rotator>
```
