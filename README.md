# \<kemet-rotator>

## Installation
```bash
npm i @kemet/kemet-rotator
```

## Usage
```js
import '@kemet/kemet-rotator/kemet-rotator.js';
```

```html
<kemet-rotator
  effect="flip"
  rotation-speed="5"
  messages='["<strong>bold</strong>", "<em>emphasis</em>", "<small>small</small>"]'>
</kemet-rotator>
```

[Click here](http://kemet.online/rotator) for more docs.
